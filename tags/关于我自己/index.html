关于 Minecraft ARG 的一些想法

定义

另类实境游戏（全称：Alternate reality game，简称ARG），是一种将真实世界作为平台的互动式剧情，通常包含多媒体与游戏元素，侵入式虚拟现实游戏的故事情节可以因参加者的想法或行动改变。Minecraft ARG，就是与《我的世界》有关的ARG，它与游戏世界的关系可以强，也可以弱。

优缺点

首先，与所有类型文娱作品一样，Minecraft ARG 作为一种限定类型的创作，肯定有其受众——最直接的就是《我的世界》玩家。出于《我的世界》玩家年龄分布较广，所以无论是成人向，还是小众黑深残的青少儿向的 ARG 都有人爱看。

ARG 最重要的两点就是真实感和沉浸式体验。但这两点，是很多 ARG 所没有做到的。

以 Wifies 的《寻找不存在之玩家》为例，它很多的线索来自于游戏内部莫名其妙出现的 Youtube 链接。在世界观内，这些 Youtube 链接是美国政府手下的一个神秘组织用来传播秘密视频的方式。我不敢想象什么样的组织会用 Youtube 传播视频，这让我想起了《绝海的侦探》里朝鲜特工用网盘传递日本国家机密。这就很不现实；就算考虑到北美目前的魔幻现实主义浪潮，也很不合理。

同样地，我们在 Wifies 的另一个 ARG《邪恶的 AI 模组》中可以看到，在 2012 年，主角在一个 Netlify 搭建的网站上下载了所谓的“MCCompanion”模组。但这根本不可能，因为 2012 年还没有 Netlify 这个东西。同样他下载模组时的 Chrome 显得非常现代，也基本上可以确定为 2020 年以后的 Chrome。另外，他的系统版本是 Windows 8.1，似乎也没有在 2012 年发行。细节上的问题多到可以影响观感的时候，这个 ARG 就可以说是失败了。

至于沉浸式体验，我个人认为这主要得体现在“可玩性”上。ARG 归根结底是 Game，玩家最好得能玩上。Zachobuilds 的《Alpha Minecraft直播》系列采取了相对新颖的方式，不是简单的在 Youtube 上堆砌视频或者纯纯藏链接。用直播演绎剧情的效果比起视频也更好，且开放 Google 账户的账户密码也可以说是大大增加了可玩性。但是这还不够。

《Alpha Ver.》才是真正可玩性高的系列。作为 RetroGamingNow 迄今为止最好的一部 ARG，不仅有相当通俗易懂的剧情，还随视频发布了数个可以游玩的 Minecraft.jar。其可玩性不下大部分《我的世界》整合包，甚至有人赞誉它是“自己玩过最好的 MC 版本”。配乐、设定、材质设计以及还原度，都属于上乘。

《BEDROCK.ZIP》在我这里有优劣二象性，一方面它的意象，文学性以及美术设计水平很高。但我认为它有些太过于云里雾里，且过于瘆人，我仍然认为它和观众的互动太少。Render 的两部 ARG，《Herobrine》和《不存在的玩家》的设计水平不高，但是剧情清晰易懂，而且让人感到清爽。不过它的可玩性也很低。

推荐

良好

  1. “You’ve never heard of these Minecraft versions. Why not?”, RetroGamingNow
    毫无争议的顶级，作为一部 Minecraft ARG，它不仅有完整的社区文化（Discord 群聊和 Wiki，还制作了可玩的版本。
  2. “This Creepy Minecraft Channel Only LIVESTREAMS”, Zachobuilds
    开创了用直播和 Google 账号与观众互动的新形式。剧情也相当生动有趣，感人至深。

    及格

  3. BEDROCK.ZIP, Zeemyth
    美术水平↑，光影水平↑，剧情↓，沉浸度（第一部视频↓，第二部视频↑），可玩性0
  4. AndrewGaming722, ThePalladium, Zeemyth
    美术水平↑，光影水平↑，剧情一般，沉浸度一般，可玩性0

佛州奇闻

前几天，我在佛州旅居的一位朋友，给我写了一封邮件。邮件里讲述了一些他和当地中国留学生的校内生活。我为之惊诧无比，同时也因其之奇趣搞笑而前仰后合。我便向他发了更多邮件问询事情的具体状况，包括一些有趣的细节。作为一个中国人，我觉得这些事情的真实度应该颇高，毕竟太过于离谱的事情，我一向认为，是编不出来的。

我的那位朋友在佛罗里达的一所高中读书，校徽是JB，倘若有心去找，说不准能找到。据他所言，他“落入这不幸的处境”，主因是受了留学中介的骗。一件幸事是那里的中国人很多，不像鲁迅在仙台留学，像“胶菜”一样有特殊待遇。不幸的是，学校将中国学生的寝室门前，都用汉字标上了名字。有别有用心者，趁没人在的时候，将假香蕉放在他们的门口，或是丢进低层的阳台。本来据说有段时间扔的是真香蕉，想必是不想用昂贵的水果便宜了中国人；故才革新了手段。
